This is the repository for the Plant Trait Ontology.

The current release version of the TO can be accessed through the official PURLs:

 * http://purl.obolibrary.org/obo/to.owl
 * http://purl.obolibrary.org/obo/to.obo
 * http://purl.obolibrary.org/obo/to/subsets/to-basic.obo

Key Ontology File Formats
- OWL (Web Ontology Language) (.owl): The standard W3C web-based language for the Semantic Web, used for complex, logic-based knowledge representation.
- OBO Format (.obo): A text-based format widely used in bioinformatics (e.g., Gene Ontology, Mondo) for its high human readability and ease of editing.

Or via http://planteome.org/
